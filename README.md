# Choro Grenoble

A web application for musicians to explore, practice, and view sheet music for Choro and Samba music. Features include sheet music display, transposing, and song/artist filtering.

## Technologies

-   **Frontend**: Vue.js 3, Vue Router, Vuex
-   **Music Notation**: OpenSheetMusicDisplay
-   **Search**: Fuse.js

## Project setup
```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run serve
```

### Compiles and minifies for production
```bash
npm run build
```
